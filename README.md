## Development

Dependencies:

- `docker`
- `docker-compose`

### Run the dev server

```sh
auto/start
```

Visit <http://localhost:1313/>.

### Dev shell

The hugo service:

```sh
auto/hugo
```

The node / wepback service:

```sh
auto/node-dev
```
